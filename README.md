# 模仿大众点评 Web APP
> 根据慕课网教学视频完成该项目，感谢王福鹏老师的讲解

该项目实践了react技术栈的相关内荣（react react-router react-redux）


## 项目结构（目录）
### app
 > 这里是项目的代码实现

- components
存放UI组件

- containers
存放容器组件

- config
 存放基本配置文件

- fetch
 将 fetch 方法封装为项目需要的存取数据的方法

- router
 配置项目路由信息（使用react-router）
 
- redux
 配置项目消息传递的信息（使用react-redux）
 
- static
 存放一些公用的静态文件
 
- util
 封装一些通用的工具类方法
 
### mock
 项目中的 mock 数据（使用koa搭建简易服务器完成）
 
### test
 测试内容
 
### UE
 效果图及素材
 
### docs
 项目中的一些文档（待整理）
